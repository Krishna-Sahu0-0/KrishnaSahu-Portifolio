## ğŸš€ Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Krishna-Sahu0-0/krishna-portfolio.git
   cd krishna-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

The build output will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## ğŸ¨ Customization

### Update Personal Information

1. **Hero Section** - Edit `src/components/Hero.jsx`
2. **About Section** - Edit `src/components/About.jsx`
3. **Skills** - Modify `src/data/skills.js`
4. **Projects** - Update `src/data/projects.js`
5. **Education** - Edit `src/data/education.js`
6. **Certifications** - Modify `src/data/certifications.js`

### Customize Colors

Edit `tailwind.config.js` to change the color scheme:

```javascript
theme: {
  extend: {
    colors: {
      primary: "#7C3AED", // Change this to your preferred color
    },
  },
}
```
